## Install
```sh
npm i @sellsuki/sellsuki-ui
```

## Usage
```html

<!doctype html>
<html lang="en-GB">
<head>
    <meta charset="utf-8">
    <title>UI Library</title>
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <script type="module" src="https://unpkg.com/@sellsuki/sellsuki-ui?module"></script>
</head>
<body>
    <div id="demo">
        <sellsuki-button>Button</sellsuki-button>
    </div>
</body>
</html>
```
