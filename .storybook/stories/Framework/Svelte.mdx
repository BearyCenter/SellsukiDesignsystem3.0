# Svelte setup

## Install

```sh
npm i @sellsuki/sellsuki-ui
```

## Usage

```html
// src/+page.svelte
<script lang="ts">
  import "sellsuki-components";
</script>

<ssk-theme-provider>
  <ssk-i18n-provider>
    <ssk-button themecolor="primary" size="md">
      <ssk-icon slot="prefix" name="solid-users" size="md" />Button
    </ssk-button>
  </ssk-i18n-provider>
</ssk-theme-provider>
```
