## Sellsuki Design System : Semantic Color Tokens

Semantic tokens reference base palette vars — ไม่ hardcode hex ค่าสีเปลี่ยนตาม brand

---

## Usage

### Option 1: ผ่าน `ssk-theme-provider` (แนะนำ)

ส่ง `brand` prop — ThemeProvider จะ inject `--palette-*` และ semantic vars บน `:root` ให้อัตโนมัติ

<Source code={`<!-- HTML / Lit -->
<ssk-theme-provider brand="patona">
  <div class="card">...</div>
</ssk-theme-provider>

<!-- เปลี่ยน brand → vars เปลี่ยนทันที -->
<ssk-theme-provider brand="ccs3">
  <div class="card">...</div>
</ssk-theme-provider>`} language="html" />

### Option 2: Manual inject (JS/TS)

ใช้เมื่อไม่ได้ใช้ `ssk-theme-provider` หรือต้องการ inject ใน root ของ app เอง

<Source code={`import { Brand, injectPaletteVars, injectSemanticTokens } from "@your-scope/sellsuki-components/theme/tokens";

// เรียกครั้งเดียวตอน app boot
injectPaletteVars();
injectSemanticTokens(Brand.patona);

// เปลี่ยน brand ภายหลัง
injectSemanticTokens(Brand.ccs3);`} language="typescript" />

### ใช้ token ใน CSS

หลัง inject แล้ว ใช้ CSS custom properties ได้ทันที — component เดิมเปลี่ยนสีตาม brand โดยไม่แก้โค้ด

<Source code={`.button-primary {
  background: var(--bg-brand-solid);       /* patona: #EC5E2A | ccs3: #32A9FF */
  color: var(--text-white);                /* #ffffff */
  border: 1px solid var(--stroke-brand-solid);
}

.button-primary:hover {
  background: var(--bg-brand-solid-hover);
}

.label {
  color: var(--text-brand-primary);
}

.card {
  background: var(--bg-primary);
  border: 1px solid var(--stroke-primary);
}`} language="css" />
